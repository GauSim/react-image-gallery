# react-image-gallery

small gallery build with react with typescript (TSX) and JSPM 

- added paging 
- added categories 
- added preloading of images 

check out the [Demo](http://apps.gausmann-media.de/react-image-gallery)

get it running: 

```
$ npm instsall
$ jspm install 
$ tsd install
$ npm start
```
